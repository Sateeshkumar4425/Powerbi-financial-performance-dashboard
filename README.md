# Financial-performance-dashboar(Interactive Dashboard creation by using Powerbi)    
## Project Objective

To build an interactive Power BI dashboard that analyzes key financial KPIs across regions, segments, and time to support data-driven business decisions.

## Dataset used
- <a href="https://github.com/Sateeshkumar4425/Powerbi-financial-performance-dashboard/blob/main/Financial%20Sample.xlsx"> Dataset </a>

##  Key KPIs
- **Total Sales**
- **Total Profit**
- **Profit Margin**
- **Units Sold**
- **Average Discount**
---
##  Business / KPI Questions Answered
- How are overall sales and profit performing?
- Which countries and segments contribute the most to revenue and profit?
- What is the impact of discounts on profitability?
- How does performance vary by year and month?
- Are high unit sales translating into higher profit?
